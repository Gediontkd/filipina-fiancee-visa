# Filipina Fiancee Visa Platform

## Project Overview
Automated immigration support platform for U.S. visa consultancy specializing in K1, CR1, and other immigration packages.

## Tech Stack
- Laravel (PHP Framework)
- MySQL Database
- AWS Infrastructure (EC2, RDS, S3)
- Bootstrap/CSS for Frontend

## Development Setup
1. Clone repository
2. Copy `.env.example` to `.env` and configure
3. Run `composer install`
4. Run `npm install`
5. Run `php artisan key:generate`
6. Import database
7. Run `php artisan serve`

## Server Information
- Dev Server: AWS EC2 Instance
- Dev URL: http://dev.filipinafianceevisa.com
